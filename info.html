Info over Studie-quiz

Algemene informatie

Studie-quiz is een website/project met het doel om leerlingen te helpen met leren door gratis quizzen online toegangelijk maakt.

Vanuit mijn ervaring werkt het leren met de aangeboden quizzen het best als je de woorden opschrijft op papier voor dat je checkt of je het goed had.

Er word niet gegarandeerd dat de quizzen overeenkomen met de schoolboeken of dat de quizzen grammaticaal/inhoudelijk kloppen. Het is je eigen verantwoordelijkheid om de juiste stof te leren. Let dus altijd op of je het juiste boek hebt gekozen en de juiste stof.

Hoe maak ik een quiz?

Op de studie-quiz homepage staat de knop "maak een quiz".

De knop leid naar een pagina waar een JSON quiz kan woorden omgezet in Base64. JSON is gegevensformaat. Er staat een voorbeeld van op de "maak een quiz" pagina. Dit voorbeeld kan in een programma zoals kladblok op je windows computer of een ander programma waarin tekst kan worden bewerkt, aangepast worden.

Wanneer je de JSON quiz af hebt, kan je hem omzetten in Base64 en gebruiken op de "Speel een quiz" pagina.

In het JSON voorbeeld staat ook "moeilijkheid", deze kan je negeren omdat het (nog) niks doet.
